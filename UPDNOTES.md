# NEW Upd v1.3.1
## Update notes:

### **Added Saber-Cut Hover Animations:**
Added an animated high-speed diagonal light-streak overlay to active interface buttons and links, mimicking a saber slicing through blocks.

### **Added Native UI Sound Effects Matrix:**
Integrated a subtle, crisp synthesizer sound system using the browser's native Web Audio API (Opt-in via navigation bar). Includes tailored audio cues for elements like clicks, hover highlights, dropdown menus, and window sweeps without needing external asset files.

### **Added Mobile Swipe Gestures & Neon Touch Trail:**
Maximised mobile user experience by introducing fluid swipe gestures—swipe horizontally to shift categories or downward to close modals and lightboxes. Paired with a hardware-accelerated overlay canvas that tracks finger dragging with a beautiful glowing neon trail matching your active theme color.

### **Removed Experimental Audio Previews:**
Cleaned up backend scripts and layout files by stripping out experimental simulated track sample previews to maintain page performance and data efficiency.
